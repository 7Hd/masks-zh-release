基本動作 #

在遊玩《蒙面少俠》的過程中，每個玩家都會使用到這些基本動作，推動故事的發展。

這七個基本動作分別是：〈直面威脅〉、〈釋放你的力量〉、〈保衛某人〉、〈評估局勢〉、〈煽動某人〉、〈安慰或支持某人〉、〈揭穿面具〉。本章將提供這些動作各自的詳細說明、以及他們如何在故事中發揮作用的使用範例。

直面威脅 #

當你直接與威脅交戰時，擲+危險。

擲出10+時，從下列挑選二項。
擲出7-9時，挑選一項。

  • 抵抗或避開對方的攻擊
  • 奪走他們身上的某物。
  • 為你的隊友創造一個機會。
  • 讓對手恐懼、吃驚、或留下深刻印象。

釋放你的力量 #

當你釋放你的力量克服阻礙、重塑環境、或延展感官時，擲+古怪。

若成功，你成功達成目的。
擲出7-9時，你標記一個狀態、或者ＧＭ會告訴你這股力量是多麼不穩定或僅只造成暫時性的影響。


保衛某人 #

當你從某個立即性的威脅中保衛某人時，擲+救助。

對於NPC造成的威脅：
若成功，你保衛了他的安全，並選擇一項。
擲出7-9時，你付出代價：你使你自己暴露在危險之中、或使局勢升級。

  • 增加一點團隊默契
  • 獲得對保衛對象的影響力
  • 清除一個狀態

對於PC造成的威脅：
若成功，你使對方的擲骰-2。擲出7-9時，你使你自己暴露在代價、報復、或批評之下。


評估局勢 #

當你評估局勢，擲+優越。

擲出10+時，詢問二項。
擲出7-9時，詢問一項。
你在依據問題的回答行動時將獲得+1加值。

  • 這裡有什麼可以讓我用來_____？
  • 這裡最大的威脅是什麼？
  • 這裡什麼正處於最大的危險之中？
  • 這裡有誰對我來說是最脆弱的？
  • 我們怎樣才能最快的結束這一切？

煽動某人 #

當你以言辭煽動某個會被你言詞影響的對象時，說出你想要從他們身上獲得的事物，並擲+優越。

對NPC：
擲出10+時，他們上鉤並做出你想要他們做的事。
擲出7-9時，他們可以改為選擇下列其中一項：

  • 他們支吾停頓：你獲得下次對付他們時+1
  • 他們犯下錯誤：你獲得一個關鍵機會
  • 他們過度反應：你獲得對他們的影響力

對PC：
擲出10+時，兩者皆有。擲出7-9時，選擇一項：

  • 如果他們照做，增加一點團隊默契
  • 如果他們不照做，他們標記一個狀態

安慰或支持某人 #

當你安慰或支持某人時，擲+平凡。

若成功，他們聽見了你的話：他們選擇獲得一點經驗值、清除一個狀態、或調動標籤屬性。
擲出10+時，你可以同時增加一點團隊默契、或清除你自己身上的一個狀態。


揭穿面具 #

當你揭開某人的面具以見識他的真面目時，擲+平凡。

擲出10+時，詢問三項。擲出7-9時，詢問一項。

  • 你打算要做什麼？
  • 我要怎麼樣才能讓你的角色去_____？
  • 我要怎麼樣才能獲得對你的影響力？
  • 你真正的計畫是什麼？
  • 你想讓我做什麼？
